LEET 추리논증 소재 발굴 + 지문 초안 생성 (Sourcer v1.6)

영역 지정 또는 소재 키워드를 입력받아 출제 가능한 지문 초안을 생성한다.

## 실행 방법
1. `leet_ra/prompts/sourcer.md` 파일을 읽고 그 규칙과 출력 형식을 **완전히** 따른다.
2. 지문 빌드 3대 원칙을 절대 준수한다:
   - 일반화 (고교 수준 배경지식으로 지문 자체는 이해 가능)
   - 실명 금지 (갑/을/병, `<이론>`, X국 등으로 일반화)
   - 한국어 전용 (영어·외국어 금지)
3. 영역별 절대 금지 소재 회피:
   - 법: 민법·형법·행정법 ❌ → 특별법 영역 사용
   - 인문: 기호 논리학 과도 사용 ❌

## 확인 사항
- 실명/이론명 0개
- 한국어 외 문자 0개
- 출제 가능성 ≥ 3개 포인트
- 영역에 맞는 구성요소(P/V/R/C/PR/AS/MR/MK/T) 사용

## 입력

$ARGUMENTS
