# RA-SOURCER: 소재 발굴 + 지문 초안 생성기

LEET 추리논증의 **병목 단계**. 영역 지정 또는 키워드를 받아 출제 가능한 지문 초안을 생성한다.
소재가 부실하면 이후 analyze/design/review가 전부 무너진다. 여기에 시간을 투자한다.

## When to Activate

- 사용자가 소재만 주고 지문부터 문항까지 만들어달라고 할 때
- 영역(법/인문/사회/과학/논쟁) 지정 요청
- 세트 단위 요청의 각 라운드 시작 시

## Workflow

1. `leet_ra/prompts/sourcer.md` 파일을 Read한다.
2. (세트 모드인 경우) 이전 라운드의 소재 키워드를 중복 방지용으로 수집한다.
3. **opus 서브에이전트를 spawn**한다:

```
Agent(
  name="ra-sourcer",
  description="LEET 추리논증 소재 발굴 + 지문 생성",
  model="opus",
  mode="auto",
  prompt="{sourcer.md 전문}\n\n## 입력\n\n{영역 지정 또는 소재 키워드}\n\n## 중복 방지\n\n{이전 소재 리스트 or 없음}"
)
```

4. 결과(지문 초안)를 사용자에게 전달하고 **승인을 받은 후** ra-analyze로 넘긴다.

## 지문 빌드 3대 원칙 (절대 규칙)

1. **일반화**: 고교 수준 배경지식으로 지문 자체는 편하게 이해 가능. 변별은 선지 판단에서 발생.
2. **실명 금지**: 마르크스·칸트·하버마스 등 실제 학자명 절대 금지. '갑', '을', '병', `<이론>`, 'X국' 등으로 일반화.
3. **한국어 전용**: 영어·외국어·학술 약어 금지.

## 영역별 절대 금지

| 영역 | 절대 금지 / 권장 소재 |
|------|--------------------|
| **법 (규범)** | 민법·형법·행정법 등 변호사시험 빈출 소재 ❌. **특별법 영역(환경/교육/소비자/전자상거래 등) 사용.** |
| **인문** | 기호 논리학(∀, ∃, →) 과도 사용 ❌. '대우' 정도만 허용. 전공 개념은 지문 내 충분한 설명 필수. |
| **사회/과학** | 특별한 금지 없음. 소재 다양화에 초점. |

## 품질 체크

- 실명/이론명 **0개** (체크 후 '갑/을/병/이론'으로 교체됨)
- 한국어 외 문자 **0개**
- 출제 가능성 ≥ 3개 포인트 (이 지문에서 뽑을 수 있는 논증 지점)
- 영역에 맞는 빌드 구성요소 사용 (P/V/R/C/PR/AS/MR/MK/T)
- 고교 배경지식으로 지문 이해 가능 (지문 내 설명 완결성)

## Notes

- sourcer는 **'환각' 허용**. 논리가 풍부하고 말만 되면 실제 법령/논쟁이 아니어도 된다. 큰 수준의 일반화 전제.
- 규정(R) 작성 시 시간 표현 명확화, '아니한다' → '않는다', '만 원' 띄어쓰기 등 세칙 준수 (sourcer.md 참조).
- 사용자 승인 없이는 ra-analyze로 넘기지 않는다.
